# 6.5 Learning Paths

## 6.5.1 Business Purpose

Learning Paths provides structured, personalized learning journeys that guide employees through progressive skill development and career advancement. This ensures systematic skill acquisition, competency building, and role readiness while supporting various learning methodologies, progress tracking, and integration with performance management and talent development systems.

## 6.5.2 Business Scope

Enable complete learning path management including path design, curriculum sequencing, progress tracking, and outcome measurement. The system supports multiple path types, adaptive learning, milestone tracking, and audit trails while ensuring accuracy, effectiveness, and alignment with organizational requirements and individual development needs.

## 6.5.11 Workflow & Approvals

1. **Path Creation** – HR defines path metadata (name, objectives, target audience, budget, approval workflow).  
2. **Module Assembly** – Curriculum designers add modules, set sequencing, prerequisites, and associate content.  
3. **Content Upload** – SMEs upload learning assets, tag metadata, and set access permissions.  
4. **Approval Routing** – Path, modules, and content flow through configurable approval chains (e.g., Learning Ops → Department Head → L&D Director).  
5. **Enrollment Trigger** – Automatic enrollment based on role, talent pool, or manager request.  
6. **Progress Monitoring** – Real‑time dashboards show enrollment counts, completion rates, and bottlenecks.  
7. **Audit Trail** – All changes, approvals, and enrollment actions are logged for compliance.

## 6.5.12 Metrics & KPIs

| KPI | Definition | Target |
|-----|------------|--------|
| **Enrollment Rate** | % of eligible employees enrolled in a path | ≥ 75 % |
| **Completion Rate** | % of enrolled participants who finish the path | ≥ 80 % |
| **Time to Completion** | Average days from enrollment to path completion | ≤ 90 days |
| **Skill Improvement Score** | Avg. increase in skill assessment scores pre‑ vs post‑path | ≥ 15 % |
| **Certification Yield** | % of participants earning a certification | ≥ 60 % |
| **Cost per Completion** | Total learning spend divided by completed enrollments | ≤ $500 |
| **Learner Satisfaction** | Survey rating of content relevance & quality | ≥ 4.0 / 5 |
| **Business Impact** | Correlation of path completion with performance rating uplift | ≥ 0.3 |

## 6.5.13 Best Practices

- **Modular Design** – Build reusable modules that can be assembled into multiple paths.  
- **Adaptive Learning** – Leverage the `adaptive_learning` flag to personalize content sequencing based on learner performance.  
- **Micro‑Learning** – Include short, focused content chunks to improve engagement and retention.  
- **Continuous Feedback** – Embed short surveys after each module to capture real‑time learner sentiment.  
- **Data Governance** – Enforce data quality checks on enrollment, progress, and assessment data.  
- **Integration** – Sync completions and certifications with the HRIS and performance management systems.  
- **Analytics‑Driven Updates** – Use the metrics above to iteratively refine curricula and retire low‑impact modules.
